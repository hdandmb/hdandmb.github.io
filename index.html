Welcome To CIT-82

CIT-82 is a beginner web development class taught at Fresno City College. It is a part of the Business Edcuation Division. It was 9 intense weeks of learning the basics of web development. It is an online, short-term class and the instructor I ended up taking is Rio Waller. In this class, I tackled web development languages, common tools used, topics, and specializations. I had never taken a web development class until this one. As part of the class I was tasked with building my very own site, having a choice of a personal site or a site on CIT-82. I don’t believe I’m that interesting, so I chose the CIT-82 site. This site, which ultimately was developed under the project name Project 0, is the result of all that hard work. I have included some of the content I found to be the most interesting that I have learned and taken from this class to share with the masses. It was a hard decision to make but the categories I have chosen to include are Front End, Web Development, Current Topics and Ecommerce. I had an amazing time taking this class, there was some things I knew and some things that I didn’t, but it was a ride from start to finish. And I’m happy that I can share all that I have done in this site.
